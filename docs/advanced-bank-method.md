# 大三大四高阶题库说明

## 定位

这套题库面向大学德语高年级学习者，训练范围接近 CEFR C1 的阅读、语言运用、翻译与书面表达能力。它是日常训练库，不是官方试题、真题汇编或完整考试模拟。

## 公开参考框架

- [Goethe-Zertifikat C1 考试训练](https://www.goethe.de/de/spr/prf/ueb/pc1.html)：阅读、听力、写作和口语模块的公开任务结构。
- [Goethe-Zertifikat C1 考试目标与测试说明](https://www.goethe.de/pro/relaunch/prf/sr/Handbuch_Pruefungsziele_Testbeschreibung_C1.pdf)：长文本理解、篇章重构、正式写作和复杂议题表达的能力说明。
- [TestDaF 官方数字考试示例](https://www.testdaf.de/de/teilnehmende/der-digitale-testdaf/vorbereitung-auf-den-digitalen-testdaf/beispielaufgaben-zum-digitalen-testdaf/)：学术阅读、信息提取、写作与中介任务的公开类型。
- [德国联邦议院职能说明](https://www.bundestag.de/parlament/aufgaben-197186)：德国政治制度题使用的稳定基础事实。
- [奥地利议会：联邦与各州](https://www.parlament.gv.at/verstehen/politisches-system/bund-laender)：奥地利联邦结构题使用的稳定基础事实。

这些资料只用于确定能力范围、题型和稳定知识事实。题干、选项、语篇、译文和解析均为本站重新编写，不复制公开示例题的文字。

## 质量状态

题库已完成自动化结构检查，包括总数、唯一 ID、必填字段、选择题答案归属、语篇引用和模块数量。开放翻译与写作采用参考答案加学习者自评，避免把合理的不同表达直接判错。

当前版本由规则模板生成并抽样检查，尚未由德语教师逐题审校。正式用于课程、收费服务或备考承诺前，应进行人工语言审校、难度标定和小规模学生试测。
