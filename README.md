# Deutsch Sprint

大学生德语词汇与语法复习网站模板。

## 当前版本

- 红白主题学习界面
- 今日学习概览
- 可选择的大一大二 / 大三大四学习阶段
- 示例单词卡片
- 分阶段语法模块占位
- AI 辅助讲解入口占位
- 朋友个人网页关联入口
- 错词复习与进度面板

## GitHub Pages 部署

1. 打开仓库的 `Settings`。
2. 进入 `Pages`。
3. Source 选择 `GitHub Actions`。
4. 保存后等待 1-3 分钟。

部署完成后，网站地址通常是：

```text
https://zhan4500-cyber.github.io/deutsch-sprint/
```
